Getting Started

Starting QLC

  • Start qlc from the desktop menu or from a terminal with the command:
    • 	qlc

    After some messages the main window will open. If there are any errors, refer to Errata and FAQ.

    Main Window

    Most controls throughout the software have tooltips, which pop up after holding your mouse button over a control item (button, slider etc.) for a while. The menubar on the top of the workspace window contains vital operations which we will discuss a bit later. Below the menubar is the toolbar that contains the following buttons (from left to right): new, open, save, device manager, virtual console, function manager, panic, blackout and mode change.

    Notice also the status bar at the bottom of the window. There are the indicators for mode and blackout. The mode indicator says "Design" which means that the application is in design (edit) mode. The blackout indicator is not visible, but when you click the blackout button on the toolbar, the indicator starts flashing to notify that all channels are disabled and their values are set to zero (until you release the blackout by pressing the button once more).

    Device Manager

    The device manager is the heart of qlc's device-oriented architecture. As its name already says, you manage your DMX devices (i.e. fixtures) from the device manager.

  • Open the device manager now by pressing its button on the toolbar.

    • The device manager also has a toolbar, containing the following buttons (from left to right): Add, Clone, Remove, Properties, Monitor and Console. In short, you can add new devices, create exact replicas clones of existing devices and their functions, edit the properties of your devices, monitor the devices' channel values and use the console to control the channel values of each device.

  • Add an input device (press the first button from left) to our workspace. You should see a the following dialog:

    • New Device

    On the left side you can see the list of available fixture manufacturers. Each manufacturer folder contains a number of fixtures produced by each manufacturer. You can find for example an "MX-4" under "Martin" folder. As you click a fixture from the list, you can see the channels field on the right change to display the number of DMX channels required by the selected fixture.

  • Go ahead and select Martin MX-4 but don't click OK just yet.

    • You can edit the new device's name in the Name field and its DMX address to the Address field. If you don't understand the DMX addressing principles, please consult your lighting equipment manuals for more info.

    Device Manager with DPX-620 as an output device

    In the left pane you can see the devices that are present in your current workspace. In the right pane, you can see the currently selected device's information.
