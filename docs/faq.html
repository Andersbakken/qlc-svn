Problem Solution
I would like to see this or that new feature in QLC. Whom should I tell? We have a special feature request database for that at Sourceforge. Same goes for bugs.
Can I use QLC on a Windows computer? Although QT-based applications such as QLC can be ported to Windows quite easily, the dmx4linux driver suite is only for Linux. There are no similar kinds of driver suites for Windows that we'd know of. If you know one, please, tell us!
I can't find my dimmer/scanner/effect in QLC's device list! Will QLC work with my equipment? Yes, You can use the device class editor (/usr/local/qlc/bin/dce) to create any kind of device you have ever wanted to control. After all, it's just a matter of channels, which are limited only by the DMX protocol to a maximum of 512.
Can I use analoque equipment (0-10 V) with QLC? Yes, you may try the i2c interface described in the dmx4linux suite.
