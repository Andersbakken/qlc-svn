Problem Solution
I would like to see this or that new feature in QLC. Whom should I tell? We have a special feature request database for that at Sourceforge. Try this link. Same holds for bugs!
Can I use QLC on a M$ Windows computer? No! and Yes! QLC is tied to Linux and the great dmx4linux driver suite. However, as Linux can be given away and copied without a charge, you make ask a friend to lean you his installation CD. Most recent Linux distributions come with automagical installers that may even free the neccessary space and create needed partitions.
I can't find my dimmer pack in QLC's device list! Will QLC work with my equipment? Yes, 99% sure about that. Dimmers are very simple devices. Just use the generic dimmer device class that fits your number of channels. Even any other brand may work.There will be some features in QLC in the future that - e.g. compensate for non-linear dimmer response and are device dependent. However, for the time beeing: don't worry about that!
Can I use analoque equipment (0-10 V) with QLC? Yes, you may try the i2c interface decribed in the dmx4linux suite. More about this very-low cost interface can be found here.
