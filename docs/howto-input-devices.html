Input Devices Howto

This howto document tells you how to associate input devices to input universes and how to edit these devices. You should first read the howto on input mapping so that you can access the input manager and know how to edit input universes.

Input Manager

Click the Inputs item from the main window's Manager menu.

Mapping Properties

Select universe 1 and then click the edit button to edit the first universe's mapping properties.

The dialog that opens is the same that has been covered in the input mapping howto. However, the dialog contains two pages: Mapping and Device that are accessed thru the little tabs at the top of the dialog.

Device assignment

Click the Device tab to access the input device tab.

You'll see a list of available input device definitions, with a check mark on None. This means that the current universe doesn't have a device assigned yet. To assign a device to the universe, simply place a check mark on one of the available devices by clicking on the empty check box. Only one device can be assigned to one universe at a time, so the check mark will actually move from its previous position to the item that you have selected.

Click the white box beside Behringer - BCF2000 to assign it to the current universe. If you're happy with this, click OK to close the dialog and accept these mappings.

Add/Edit a Device

It is probable that your input device is not on the list and you need to create one of your own.

Click the create new input device button to start making a device definition for your input device. Alternatively, you can edit any existing device by selecting the appropriate item and clicking the edit button. The procedure is exactly the same in both cases from now on.

Input Device Editor

A dialog is opened with entry fields for Manufacturer and Model.

Enter the device's manufacturer and model to these fields and then click the Channels tab to edit the device's channels.

Now you have two choices for adding channel definitions: Manual and Automatic:

Click the add button to enter individual channel information by hand for each channel.
Click the automatic wizard button to attempt automatic channel detection. You'll receive further instructions from QLC. You must have an input device assigned to the current universe for this feature to work. Also, you must first stop the wizard to be able to navigate away from this dialog page.

You can edit the existing channels by selecting them and clicking the edit button. To remove selected channels, click the remove button.

When you add or edit a channel, you must enter the channel number, name and type (Slider, Knob or Button). Note that you cannot add the same channel multiple times to one device.

When you're ready, click the OK button to accept changes and save the input device. If you didn't enter a manufacturer/model for the device, you'll be prompted to enter them before you can continue.

Back in the mapping properties dialog

Now you should see the device you just defined in the list of available input devices. Remember how to assign it to the current universe? Scroll up to Device assignment if you don't.

To remove any existing input devices, click the remove button. Note that some devices are so-called system devices and cannot be removed unless you're the administrator.

Click OK to accept these mappings and close the dialog.
