Bus List Columns
Number: Bus ID number.
Name: Bus name; always use descriptive names.

Left-click on the name column will let you give a name to the selected bus. A total of 32 buses are available for function speed setting. Although the same bus can be assigned to any function (see virtual console and function tree for more info on bus assignment), each function type interprets the same bus value differently:
 
Scene understands the value as fade time
Chaser understands the same value as hold time, whereas
Collection merely ignores all bus values.

The first two buses cannot be renamed. Default Fade is assigned to all newly created scene functions, and the second, Default Hold is assigned to all newly created chaser functions.
