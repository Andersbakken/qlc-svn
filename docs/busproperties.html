Bus List Columns

Number: Bus ID number.
Name: Bus name; always use descriptive names.

Left-click on the name column will let you give a name to the selected bus. A total of 32 buses are available for function speed setting. Although the same bus can be assigned to any function, each function type understands bus values differently:

    Scene: fade time (time given to fade from level A to level B)
    EFX: duration of one cycle (ex. a full 360-degree circle)
    Sequence: hold time between steps
    Chaser: hold time between steps
    Collection: no effect

A bus is assigned to each newly created function:

    Scene: Default Fade (01)
    EFX: Default Fade (01)
    Sequence: Default Hold (02)
    Chaser: Default Hold (02)
    Collection: None

You can change the bus assignments for each function from the Function Manager.
