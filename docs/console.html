Console


Click to see bigger

Elements

The console is the lowest-level of control in QLC that you can have over a fixture. The left side of the console is reserved for a very powerful scene editor while the right side contains sliders for each channel in the selected fixture.

Scene Editor

The scene editor consists of a menu button and a list of scenes created for the selected device. Clicking the menu button (or right mouse button on a scene name) displays a popup menu from which you can add/remove/edit the scenes. There is also a save status indicator below the scene list, which tells you whether you have modified the current scene (and need to create a new scene or overwrite an existing scene).

Scene Editor Menu Items

    Activate selected: Activate the selected scene (instantly sets the scene values and sliders).
    Create New: Create a new scene using the current slider values
    Overwrite: Save current slider values to the selected scene
    Remove selected: Remove the selected scene from the device.
    Rename selected: Rename the selected scene

Console Sliders

Each channel occupied by the selected device has its own slider in the console. Moving any one of the sliders will immediately set the channel value to DMX hardware (if any) unless blackout is enabled. Below each slider, there is a button, that has three states:

    Fade: The value is treated as a "fade" value in scenes
    Set: The value is treated as a "set" value in scenes
    X (NoSet): The channel value is not used in scenes

You can read more about the states from the QLC Concept page.

Console Slider Menus

Clicking the right mouse button over a console slider opens a popup menu from which you can select different capabilities (presets) defined for each channel (ex. Gobo or Color selection).

Closing

To close the window just click the close button in the upper-right corner of the window.
