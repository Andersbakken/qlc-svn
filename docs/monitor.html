Monitor


Click to see bigger

Elements

The monitor is a very simple widget that allows the user to view the values in each channel occupied by a fixture. The view contains pairs of channels and values; the channel value being just below the channel number.

Context Menu Items

Clicking the right mouse button on top of a monitor window displays a popup menu from which you can configure some aspects of the monitor. Note that changing a property in one monitor window will propagate the change to all monitor windows.

    Channel Display

      Absolute: Display absolute DMX channel numbers (i.e. start from the device's address)
      Relative: Display DMX channel numbers relative to the device (i.e. start from 1)

    Update speed

      16Hz: Update the monitor 16 times per second
      32Hz: Update the monitor 32 times per second
      64Hz: Update the monitor 64 times per second

    Resize to default: Resize the monitor to the default square aspect.
    Change Font: Change the font used to display the channels and values

Closing

To close the window just click the close button in the upper-right corner of the window.
