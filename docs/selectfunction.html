Select Function(s)

Whenever a Function needs to be selected, for example when you're adding steps to a Chaser, using the Chaser Editor, the Select Function dialog is used. You can also create new functions with this dialog, but the editing capabilities are more restricted than those of the Function Manager.

Sometimes the place you are selecting the function to, might allow selecting multiple functions, for example, when adding steps to a Chaser or a Collection. In those cases, the order in which the functions are selected will also be the order in which they are added to their destination. Oppositely, when attaching a function to a button in the Virtual Console, you may select only one function at a time.
