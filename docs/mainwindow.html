Menubar

File Menu

New: Discard all previous data and start a new workspace.
Open: Open an existing qlc workspace file (*.qlc).
Save: Save the current workspace to a file.
Save As: Specify a name to the current workspace and save it to a file.
Settings: Edit some common settings.
Exit: Exit the program (disabled in Operate Mode!)

Tools Menu

Device Manager: Open the Device Manager window.
Virtual Console: Open the Virtual Console window.
Function Manager: Open the Function Manager window.
Bus Properties: Open the Bus Properties window.
Panic: Stop all running functions in case something disastrous has happened.
Mode: Change the program's operating modes between Design and Operate.

Window Menu

Tile: Arrange visible editor windows in a tiled pattern
Cascade: Arrange visible editor windows in a cascaded pattern
 
This menu also contains a list of all open windows.

Help Menu

Index: Opens the QLC Help/Document Browser
About: Displays an about box that contains information about the program.
About QT: Displays an about box containing information about QT.
Show Tooltips: Toggle global tooltip popups on/off.

Toolbar

New: Discard all previous data and start a new workspace.
Open: Open an existing qlc workspace file (*.qlc).
Save: Save the current workspace to a file.
Device Manager: Open the Device Manager window.
Virtual Console: Open the Virtual Console window.
Panic: Stop all running functions in case something disastrous has happened.
Blackout: Set all channel values to zero; value changes are not updated to devices.
Mode: Switch the program's operating modes between Design and Operate.

Statusbar

The status bar at the bottom of the window displays some common information about the program, for example the current operating mode and when blackout is enabled, a flashing blackout indicator can be seen in the status bar.
