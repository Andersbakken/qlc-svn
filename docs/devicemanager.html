Device Manager

The device manager is divided into three main elements: toolbar, device list and device information.


Click to see bigger

Elements

Toolbar

* Add: Add a new device.
* Clone: Create a clone of the entire selected device along with its functions.
* Remove: Remove the selected device (removes also the device's functions).
* Properties: Edit the selected device's properties (name and address).
* Monitor: Open the DMX monitor to see what values are set to the device's channels.
* Console: Open a console that can be used to edit scenes to the selected device.

Device List

The device list contains a list of currently present devices in your setup. Selecting a device from this list displays the selected device's information on the device information field to the right. You can also perform various editing operations to the selected item thru a context menu by pressing the right mouse button over a device. The same actions in the context menu are available also in the toolbar.

Device Information

The device information field shows some essential information on the currently selected device, such as the device's make and model, its DMX address and the device's exact channel setup.

Closing

To close the window just click on the Close button in the upper right corner of the device manager window.
