Fixture Editor


Click to see bigger

Menubar

File Menu

New: Create a new fixture
Open: Open a fixture file from disk
Save: Save the current fixture to a file
Close: Close the currently selected fixture
Exit: Exit Fixture Editor

Window Menu

Tile: Arrange visible editor windows in a tiled pattern.
Cascade: Arrange visible editor windows in a cascaded pattern.
 
This menu also contains a list of all open windows.

Help Menu

About: Displays an about box that contains information about the program.
About QT: Displays an about box that contains information about QT.

Toolbar

New: See New in file menu
Open: See Open in file menu
Save: See Save in file menu

General Information

Manufacturer: The name of the fixture's manufacturer (e.g. Martin)
Model: The fixture's model (e.g. MAC550)
Name: The type of the fixture (scanner, flower, moving head etc.)

Channels

The channels list holds all the channels that this fixture understands. For a 12-channel dimmer you might define 12 channels named "Channel 1", "Channel 2", "Channel 3" ... all the way up to "Channel 12".

Buttons

Add: Add a new channel (the channel is added to the end)
Remove: Remove the selected channel (channels after the removed channel are raised to fill the gap)
Edit: Edit the selected channel's description
Raise: Raise the selected channel up by one (i.e. swap the places between the selected channel and the channel above)
Lower: Lower the selected channel down by one (i.e. swap the places between the selected channel and the channel below)

Capabilities

The capabilities list shows the capabilities of the selected channel. An example of a capability could be the value (or a value range) that is used to set green color in a scanner's color channel.

Buttons

Add: Add a new capability to the selected channel
Remove: Remove the selected capability (possible gaps are not filled)
Edit: Edit the selected capability's minimum and maximum values as well as the description
