Function Manager

With the function manager you can manage all of your functions that do the actual work to automate your lights. You can create new functions, remove and edit existing ones as well as create copies of them. Controls are on the upper part of the window; the lower part displays all of your functions.

Controls

Create a new Scene using the Scene Editor.
Create a new Chaser using the Chaser Editor.
Create a new EFX using the EFX Editor.
Create a new Collection using the Collection Editor.
Start the Function Wizard.
Edit the currently selected function. See Scene Editor, Chaser Editor, EFX Editor and Collection Editor for details on the specific editor, that is automatically chosen by QLC for each function type.
Create a copy of each of the selected functions.
Permanently remove the selected function(s).
Assign a bus to the selected function(s).
