Menubar

Add Menu
* Scene: Create a new scene to the selected device (available only for devices).
* Chaser: Create a new chaser to the global function space (available only to global).
* Collection: Create a new collection to the global function space (available only to global).

Edit Menu
* Cut: Prepare the selected function to be moved to another device or global.
* Copy: Prepare the selected function to be copied to another device or global.
* Paste: Move or copy the prepared function to the selected device or global.
* Delete: Delete the selected function.
* Edit: Edit the selected function; opens a separate dialog.
* Assign Bus: Assign a speed bus for the selected function.

A combination of these two menus is available by right-clicking an item in the function list.

Function List

The function list is arranged into a tree, where devices serve as the parent nodes and individual functions as the leaf nodes.

Setting the Keep device nodes open-checkbox on, all device nodes are kept always open.

Selecting a Function

The same function manager dialog is used also when selecting member functions for another function (e.g. chaser and collection).

The difference between normal and selection mode is that the selection dialog doesn't have any menus nor edit capabilities available. Also, OK and Cancel buttons are available in selection mode to accept or reject function selection.

Closing

To close the dialog, click on the close button or either one of the OK and Cancel buttons, whichever available.
