EFX Editor

The EFX editor, as its name suggests, is used to edit EFX functions. The dialog is split in three tabs:

  • The first, General tab is for selecting Fixtures, the speed bus and the fixture order.
  • The second, Movement tab is for selecting details on how the fixtures should move their heads/mirrors.
  • The third, Initialization tab is used to tell how the fixtures should initialize themselves prior to actually moving and how they should de-initialize themselves after their movement is complete.

General Tab Controls

EFX name Change the name of the EFX.
Fixture list
  • Step: Shows the order in which the fixtures start their movement in serial order
  • Fixture: The name of each fixture that has been added to the EFX
  • Reverse: You can select those fixtures, that should reverse their movement (opposite direction to non-reversed fixtures). This setting overrides the Direction setting on the Movement tab.
Add a Fixture to the EFX, after the currently selected position. Since EFX is used for moving the head or mirror of an intelligent fixture, only fixtures with pan and tilt capability (8bit or 16bit) can be added to an EFX.
Remove the selected fixtures from the EFX.
Raise the selected fixture upwards (earlier) in serial order.
Lower the selected fixture downwards (later) in serial order.
Fixture order
  • Parallel: Make all fixtures move in concert with others.
  • Serial: Each fixture starts movement a bit after the previous one, resulting in nice asymmetric fan-like movement pattern.
Speed bus Change the bus used to adjust the speed of the EFX.

Movement Tab Controls

Pattern area Shows a 2D-projection of the fixtures' head/mirror movement. Every time you change a parameter on this page, a small dot travels the complete path in the direction selected with the Direction buttons.
Direction
  • Forward: The fixtures move forwards along the pattern path
  • Backward: The fixtures move backwards along the pattern path
Run order
  • Loop: Run thru the steps over and over again.
  • Single Shot: Run thru the steps once and then stop.
  • Ping Pong: Run thru the steps over and over again, reversing direction at both ends.
Pattern Select the movement pattern algorithm.
Parameters
  • Width: Choose the pan width (0-255)
  • Height: Choose the tilt height (0-255)
  • X Offset: Move the pattern's horizontal (pan) centerpoint (0-255)
  • Y Offset: Move the pattern's vertical (tilt) centerpoint (0-255)
  • Rotation: Rotate the pattern along its axis (0-360 degrees)
  • X Frequency: Change the Lissajous pattern's X (horizontal) frequency (0-5)
  • Y Frequency: Change the Lissajous pattern's Y (vertical) frequency (0-5)
  • X Phase: Change the Lissajous pattern's X (horizontal) phase (0-360 degrees)
  • Y Phase: Change the Lissajous pattern's Y (vertical) phase (0-360 degrees)

Initialization Tab Controls

Initialize fixtures If checked, fixtures are initialized using values from a scene selected from the Scene list. When serial mode has been selected from the General tab, each fixture is initialized separately just before starting their movement. When parallel mode has been selected, all fixtures are initialized simultaneously when movement starts. NOTE: If the selected scene does not contain values affecting the EFX's fixtures, no initialization occurs.
De-initialize fixtures If checked, fixtures are de-initialized using values from a scene selected from the Scene list. The de-initialization scene is run for each fixture when the fixture stops movement. When Single-shot mode has been selected, the de-initialization scene is run automatically after each fixture has finished its movement. However, if either Loop or Ping-pong mode is selected, the fixtures stop movement ONLY when the user explicitly stops the EFX. NOTE: If the selected scene does not contain values affecting the EFX's fixtures, no de-initialization occurs.
