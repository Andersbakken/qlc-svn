Elements

Scene Name: Always use descriptive names for functions.

Channel List Columns

Number: Channel number that is relative to device.
Channel: Channel name (the purpose of this particular channel).
Preset: A preset value corresponding to a certain value range.
Value: The DMX value (0-255) for the channel in this scene.
Type: Can be Fade, Set or NoSet (see below).

The type field can be used to set how the value assigned to the particular channel is to be treated when the scene is run. If the type is Fade, the value is faded in the given time from the channel's current value to the target value defined in this scene. Set type values are namely set instantaneously, regardless of the fade time. NoSet type values are just ignored, thus this scene would not touch that channel at all.

The preset value is used to define a named purpose for a particular value range. In a scanner-type device a preset could be named as "ZigZag Gobo" that could correspond to a value range 16-20 on channel 3.

Editing

Context Menu

You can edit the preset, value and type fields by right-clicking over the appropriate column on a particular channel and selecting a value from the popup-menu.

Keyboard

You can edit the value and type fields also by left-clicking on them in the same manner and writing the value with your keyboard.

Edit Dialog

The third way is to either double-click anywhere on a particular channel, or first selecting a channel with left-click and pressing the Edit Channel button.

Closing

After you have modified the scene, you can close the dialog by clicking OK that also saves the modifications to the scene. Pressing Cancel will discard any changes and close the dialog. Apply just saves the values but doesn't close the dialog.

Closing the dialog from the window manager handle behaves like pressing Cancel.
