Elements

Directories

Background Wallpaper: QLC's (and DCE's) main window background image file name.

User Interface

Widget Style: The general style that is used to draw QLC's widgets. The available options depend on your QT library version.

Startup

Open Last Workspace: If checked, last opened/used workspace file is opened the next time QLC is started.

Lighting Control Output

Active Plugin: The output plugin that is used to write actual (DMX) data to your lighting equipment. Dummy Output is a default plugin for offline editing that does actually nothing.

Advanced Settings

All QLC settings are stored as regular key = value pairs. This list shows all settings as such, from which they can be edited (only if you know what you are doing!)

Closing

To accept changes, close the window by clicking on the OK button. To reject changes, close the window by clicking Cancel button.
